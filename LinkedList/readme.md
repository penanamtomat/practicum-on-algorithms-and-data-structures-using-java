## Linked List
what is linked list??

Linked List adalah struktur data yang digunakan untuk menyimpan sejumlah objek data biasanya secara terurut serta terhubung satu sama lain menggunakan suatu penunjuk atau pointer. Setiap elemen dalam linked list disebut node, dan setiap node terdiri dari 2 komponen utama, yaitu data dan pointer.
